## 基于Asp.Net Core

组件基于Asp.Net Core框架构建，致力于快速构建一个简单的轻量主机服务

## 遵循DDD领域驱动设计

将整个项目划分为基础域、通用域、核心域和应用域四个部分

### 基础域

负责基础组件的引入及组件抽象定义

### 通用域

负责公共模块定义与功能实现

### 核心域

负责业务层核心架构的建立与基础方法的实现

### 应用域

负责业务层应用架构的建立与基础方法的实现
